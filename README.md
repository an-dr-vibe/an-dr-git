# an-dr-git

Cross-platform Git client focused on a clear repository tree, branches, push/pull, and a diff view built from native `git diff` output.

## Development

The repo now includes the Phase 0.1 tooling scaffold for Electron, React, Vite, TypeScript, ESLint, and Vitest.

Use Node `20.9+` with `npm`.

Local setup:

- `npm install`
- `npm run lint`
- `npm run typecheck`
- `npm run test:unit`
- `npm run test:integration`
- `pwsh ./scripts/quality-gate.ps1`
